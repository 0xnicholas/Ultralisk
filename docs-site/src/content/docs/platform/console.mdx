---
title: 统一控制台
description: Ultralisk 统一控制台能力说明
---

统一控制台是 Ultralisk 平台的核心入口，帮助企业一站式管理 AI 数据中心的资源、集群、模型服务与推理 API。

## 规划中的能力

- **资源概览**：集群、节点、GPU、Workload 状态一览
- **告警中心**：统一汇聚 Prometheus / Alertmanager 告警
- **模型服务管理**：模型版本、部署状态、灰度发布
- **配额与成本**：按项目/部门查看用量与成本分摊
- **审计日志**：基于 `request_id` 的全链路追溯

## 当前状态

统一控制台仍在产品化开发中。当前已开源的 **Ultralisk Core** 可通过 API 与 Prometheus/Grafana 提供等价能力，适合已有运维团队的用户先行接入。

## 与 Core 的关系

控制台将调用 Ultralisk Core 暴露的推理 API 与指标数据，在上层提供可视化与治理功能。Core 可独立运行，不依赖控制台。
