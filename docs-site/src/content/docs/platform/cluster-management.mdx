---
title: 集群管理
description: Ultralisk 集群管理能力说明
---

Ultralisk 计划提供对 K8s 与 Slurm 等集群编排系统的统一管理能力。

## 规划中的能力

- **自动发现**：扫描并接入现有 K8s/Slurm 集群
- **节点纳管**：GPU 节点状态、利用率、故障告警
- **工作负载调度**：训练与推理 workload 的统一调度
- **弹性伸缩**：根据队列长度与资源利用率自动扩缩容
- **网络与存储**：RDMA、共享存储的配置模板

## 当前状态

集群编排能力处于路线图阶段，尚未在产品中交付。当前可通过标准 K8s/Slurm 运维方式部署 Ultralisk Core，平台未来将逐步纳管这些集群。

## 推荐实践

在集群管理功能成熟前：

1. 使用现有 K8s/Slurm 承载推理引擎
2. 在集群入口部署 Ultralisk Core 提供统一 API
3. 使用 Prometheus/Grafana 监控集群与推理服务
