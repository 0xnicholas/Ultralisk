Ultralisk 平台计划提供从模型仓库到线上推理服务的全生命周期管理。

## 规划中的能力

- **模型仓库**：统一管理模型版本与元数据
- **部署上线**：一键将模型部署为推理服务
- **灰度发布**：多版本并行、流量切分、回滚
- **自动扩缩容**：基于并发与队列长度动态调整实例数
- **服务治理**：健康检查、熔断、优雅滚动更新

## 当前已交付

当前已开源的 **Ultralisk Core** 支持通过 OpenAI 兼容接口接入以下推理引擎：

| 引擎 | 非流式 | 流式（SSE） |
|---|---|---|
| vLLM | ✅ | ✅ |
| TGI | ✅ | ✅ |
| SGLang | ✅ | 待验证 |
| Mock | ✅ | ✅ |

模型服务的生命周期管理（上线、回滚、灰度）目前需要配合现有运维流程或 K8s 实现。
