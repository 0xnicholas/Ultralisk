Ultralisk Core 提供与 OpenAI 兼容的 Chat Completions API，便于现有应用无缝迁移。

## 基础信息

- **Base URL**：`http://localhost:8000/v1`
- **认证方式**：通过 Kong 网关时携带 `api-key` header；直接访问 Core 时无需认证
- **内容类型**：`application/json`

## Chat Completions

```http
POST /v1/chat/completions
```

### 请求体

```json
{
  "model": "mock-model-0.1",
  "messages": [
    { "role": "user", "content": "hello" }
  ],
  "stream": false
}
```

### 响应

```json
{
  "id": "chatcmpl-xxx",
  "object": "chat.completion",
  "created": 1700000000,
  "model": "mock-model-0.1",
  "choices": [
    {
      "index": 0,
      "message": { "role": "assistant", "content": "Hello!" },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 1,
    "completion_tokens": 2,
    "total_tokens": 3
  }
}
```

## 流式响应

设置 `stream: true` 后，接口通过 SSE 逐句返回 token。

## 错误码

| 状态码 | 说明 |
|---|---|
| 401 | API Key 无效或缺失 |
| 429 | 请求过快或 token 配额超限 |
| 500 | 推理或内部错误 |
