Ultralisk 支持多种部署形态，从本地开发到生产级私有化数据中心。

## 部署形态

### 本地开发

- 使用 Mock 推理后端
- 不依赖 Redis / PostgreSQL / Kong
- 适合功能验证与单元测试

### 容器化部署

- `docker-compose up -d` 启动完整基础设施
- 包含 Kong、Prometheus、Grafana、Alertmanager
- 适合小规模生产或 PoC

### 私有化数据中心

- 在自有数据中心或私有云中部署
- 支持裸金属 GPU 服务器、K8s/Slurm 集群
- 标准化交付流程，专业服务团队支持

## 部署前置条件

- 确定推理引擎（vLLM / TGI / SGLang）
- 确定配额存储后端（memory / redis / redis_pg）
- 确定是否启用内容安全审核模型
- 确定是否启用 OTel 追踪
