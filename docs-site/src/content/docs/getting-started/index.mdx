Ultralisk 是面向自建 AI 数据中心的统一管控平台。当前已开源的 **Ultralisk Core** 提供了生产级 LLM API 基础设施层的核心能力，而上层平台能力（控制台、集群编排、模型生命周期管理等）正在持续演进。

## 已交付能力

- **鉴权与限流**：API Key 校验、QPS 限流、token 用量配额
- **可观测性**：Prometheus `/metrics`、TTFT/TPOT/GPU 等指标
- **内容安全**：规则引擎 + 审核模型，输入/输出双侧防护
- **日志与追踪**：JSON 结构化日志、`request_id`、OTel 追踪

## 尚未成熟的能力（路线图）

- 硬件与网络纳管
- K8s/Slurm 集群编排
- 模型生命周期管理
- 统一控制台与多租户
- 成本与配额治理

本站点会明确标注每项能力的实现状态，避免与产品现状脱节。
