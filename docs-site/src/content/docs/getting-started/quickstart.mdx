title: 快速开始
description: 5 分钟在本地跑通 Ultralisk 全链路
---

本指南将带你从源码启动 Ultralisk，使用 Mock 推理后端验证鉴权、配额、内容安全与日志追踪。

## 环境准备

- Python 3.12+
- （可选）Redis、PostgreSQL，用于生产级配额存储

## 安装与测试

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
pytest -q                              # 53 tests, < 2s
```

## 启动 Mock 后端

```bash
uvicorn app.main:app --reload --port 8000
```

## 发送测试请求

### 非流式

```bash
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"messages":[{"role":"user","content":"hello"}]}'
```

### 流式（SSE）

```bash
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"messages":[{"role":"user","content":"hello"}],"stream":true}'
```

## 接入真实 vLLM

```bash
INFERENCE_BACKEND=vllm \
INFERENCE_BASE_URL=http://localhost:8001 \
MODEL_NAME=Qwen/Qwen2.5-7B-Instruct \
uvicorn app.main:app --port 8000
```

## 启用完整基础设施

```bash
docker-compose up -d
```

服务端口：

- `8080` Kong 代理入口（鉴权 + QPS 限流）
- `9090` Prometheus
- `3000` Grafana（默认 admin/admin）
- `8001` Kong Admin API
